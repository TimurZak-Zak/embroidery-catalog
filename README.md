# Embroidery Catalog

Онлайн-каталог дизайнов для приложения **Embroidery Studio**.
Файл `catalog.json` подтягивается приложением и формирует ленту библиотеки.

Текущий набор — временные seed-дизайны (простые формы, общественное достояние),
чтобы приложение работало end-to-end. Реальная кураторская подборка из легальных
бесплатных источников (CC0 / public domain) добавляется позже.

- `catalog.json` — список дизайнов
- `designs/*.vp3` — файлы вышивки (формат Husqvarna Viking)
- `thumbs/*.png` — превью
- `generate.py` — генератор seed-набора
